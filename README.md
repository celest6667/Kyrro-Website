# Kyrro — Site officiel

Site officiel de **Kyrro**, studio d'automatisation et d'IA appliquée basé à Lille.

> On comprend votre métier, on conçoit vos outils, on automatise vos process.

---

## ✨ À propos

Kyrro accompagne les entreprises dans la conception de leurs outils internes, l'automatisation de leurs process et l'intégration de l'IA là où elle apporte une vraie valeur. Le site présente l'approche, les offres, la méthode et les réalisations du studio.

## 🛠 Stack technique

Site statique, sans framework, pour rester rapide, simple à maintenir et à déployer partout.

- **HTML5** sémantique
- **CSS3** (variables CSS, grid, animations)
- **JavaScript** vanilla (ES6+, sans dépendance)
- **Polices** : Cabinet Grotesk · Instrument Serif · DM Sans (via Google Fonts)

## 📁 Structure

```
kyrro-studio/
├── index.html      # Structure et contenu du site
├── style.css       # Styles complets
├── script.js       # Interactions (loader, curseur, reveal, count-up)
└── README.md
```

## 🚀 Lancer en local

Aucun build, aucune dépendance. Il suffit d'ouvrir `index.html` dans un navigateur.

Pour un confort de développement (rechargement, serveur local), tu peux utiliser n'importe quel petit serveur statique :

```bash
# Python
python3 -m http.server 8000

# Node (si tu as npx)
npx serve .
```

Puis ouvrir [http://localhost:8000](http://localhost:8000).

## 🌍 Déploiement

Le site étant statique, il peut être déployé sur n'importe quelle plateforme :

- **GitHub Pages** : activer Pages dans les paramètres du repo
- **Vercel** : `vercel` à la racine, déploiement instantané
- **Netlify** : drag & drop du dossier
